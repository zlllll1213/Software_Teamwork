# 技术监督辅助平台需求说明书

## 1. 概述

本系统旨在建设一个面向电力行业的智能化服务平台，整合知识管理、知识问答与报告生成三大核心能力，为用户提供从知识沉淀、知识检索到知识应用的完整闭环。

---

## 2. 知识管理

### 2.1 知识库管理

- 系统应支持创建、编辑、删除知识库，每个知识库具有独立的名称和描述。
- 知识库应支持按文档类型进行分类，包括但不限于：
  - 规程规范
  - 技术报告论文
  - 术语条目
  - 通用文档
- 知识库应支持配置文档分段策略，至少支持：
  - 基于标题层级智能分段
  - 按固定字符数分段
- 固定长度分段模式下，应支持配置：
  - 切片长度
  - 重叠长度
  - 递归合并模式
  - 自定义分隔符
- 知识库应支持配置检索策略，至少支持：
  - 语义向量检索
  - 向量检索 + 重排序
- 知识库应支持展示元信息：
  - 文档数量
  - 创建时间
  - 创建人
- 知识库列表应支持：
  - 按类型过滤
  - 按关键词搜索
- 知识库应支持批量删除操作。

### 2.2 文档管理

- 系统应支持向知识库上传文档，支持格式包括但不限于：
  - PDF
  - DOCX
  - PPTX
  - XLSX
  - MD
  - TXT
  - 常见图片格式
- 上传文档时支持添加标签，用于后续检索过滤。
- 支持：
  - 拖拽上传
  - 点击选择文件上传
- 支持文档处理全流程状态跟踪：
  - 已上传
  - 解析中
  - 切片中
  - 向量化中
  - 就绪
  - 失败
- 处理失败的文档应支持：
  - 查看错误原因
  - 重试处理
- 支持：
  - 单个删除
  - 批量删除
- 就绪状态文档支持查看切片详情。
- 文档列表支持：
  - 分页浏览
  - 按状态筛选
- 文档支持编辑标签信息。
- 当知识库分段策略或检索策略变更时，应自动触发后台重新处理所有就绪状态文档。

### 2.3 文档解析与处理

- 系统应支持将上传文档解析为结构化内容。
- 根据知识库分段策略进行语义切片。
- 支持将切片内容进行向量化嵌入并持久化存储至向量数据库。
- 文档处理管线支持并发控制，可配置最大并发解析任务数。
- 支持多种文档解析后端，并允许运行时切换。

### 2.4 素材管理

- 系统应支持电厂专业素材文档上传和管理。
- 素材分类至少包括：
  - 厂级专业报告
  - 技术文档
  - 检查报告
- 素材支持标签化管理，通过键值对标签实现多维度分类与检索。
- 素材列表支持标签组合条件过滤。
- 支持编辑标签信息：
  - 增加标签
  - 删除标签
  - 修改标签
- 素材支持删除操作。

### 2.5 系统配置

#### 嵌入模型配置

支持运行时配置：

- 模型名称
- API 地址
- 向量维度

#### 重排序模型配置

支持运行时配置：

- 模型名称
- API 地址
- Top N

#### 文档解析器配置

支持运行时配置：

- 并发数
- 解析后端选择

#### 大语言模型配置

支持运行时配置：

- API 地址
- 密钥
- 模型名称
- 超时时间

#### 知识问答配置

支持运行时配置：

- 术语知识库选择
- 技术监督知识库选择
- 检索参数
  - Top K
  - 相似度阈值
  - 重排序阈值

## 3. 知识问答

### 3.1 智能对话

- 提供基于大语言模型的智能对话能力。
- 支持自然语言交互。
- 支持流式输出（SSE）。
- 支持非流式输出。
- 支持多轮对话管理。
- 用户侧支持：
  - 创建新会话
  - 切换历史会话
  - 删除会话
- 会话历史支持本地持久化存储。
- 页面刷新后自动恢复历史会话。

### 3.2 意图识别与路由

- 支持自动识别用户意图。
- 根据意图路由至对应处理模块。

支持的意图包括但不限于：

- 知识问答
- 数据分析
- 报告生成
- 一般对话

支持基于用户权限的意图访问控制，不同角色拥有不同功能访问权限。

### 3.3 知识增强检索

- 支持基于 RAG（检索增强生成）的知识问答。
- 支持跨多个知识库联合检索。
- 支持：
  - 语义向量检索
  - 向量检索 + 重排序
- 支持：
  - 相似度阈值过滤
  - 重排序分数阈值过滤
- 支持基于文档标签的元数据过滤。
- 检索结果 Top K 数量可配置。

### 3.4 引用溯源

- 回答支持引用标注。
- 用户可查看：
  - 原文片段
  - 来源文档名称
  - 所在章节路径
  - 相关性分数
- 多个连续引用来自同一文档时支持合并显示。
- 引用弹窗支持：
  - 悬浮预览
  - 点击固定
- 支持根据文档 ID 下载原始文档。
- 支持长文本滚动预览。
- 展示切片类型信息。

### 3.5 思考过程展示

- 支持展示 AI 多步思考过程。
- 思考步骤支持：
  - 折叠
  - 展开
- 默认在流式完成后自动折叠。
- 支持区分：
  - 代码内容
  - 文本内容

### 3.6 检索体验管理

管理员支持通过专用页面测试知识库召回效果。

检索测试支持：

- 选择一个或多个知识库
- 配置检索参数：
  - Top K
  - 是否启用重排序
  - 相似度阈值
  - 重排序阈值
  - 标签过滤

结果展示：

- 文档名称
- 章节路径
- 相关度分数
- 内容预览

## 4. 报告生成

### 4.1 报告大纲生成

- 用户输入报告主题后，AI 自动生成报告大纲。
- 支持多级章节结构：
  - 章
  - 节
  - 子节
- 支持特定报告模板驱动：
  - 迎峰度夏检查报告
  - 煤库存审计报告
  - 等
- 大纲支持在线编辑：
  - 删除章节
  - 调整顺序
- 编辑后自动重新编号。
- 每个章节应标识类型：
  - 正文
  - 表格
  - 图片

### 4.2 报告内容生成

- 根据大纲逐章节流式生成内容。
- 实时查看生成进度。
- 展示总体进度：

```text
已完成章节数 / 总章节数
```

- 每章节内容支持在线编辑。
- 支持表格与图片内容处理。

### 4.3 报告导出

- 支持导出 DOCX 文档。
- 保留用户编辑修改内容。
- 导出文档遵循统一样式体系，包括：

  - 字体
  - 字号
  - 标题层级
  - 段落间距
  - 表格样式
  - 页眉页脚
- 支持基于已保存报告数据重新生成 Word。
- 无需重新执行 AI 生成。
- 用户可下载生成的报告文件。

### 4.4 报告记录管理

记录内容包括：

- 报告名称
- 类型
- 专业
- 电厂
- 年份
- 生成时间
- 状态

支持：

- 分页浏览
- 查看历史记录
- 重新生成 Word
- 重新下载文件
- 删除记录

支持图表编号模式切换：

- 全局编号
- 按章编号

### 4.5 报告模板管理

- 支持上传和管理模板文件。
- 模板支持关联素材文档。
- 建立模板与素材映射关系。
- 支持基于映射关系进行素材检索与管理。
- 模板支持删除。
- 模板支持可视化编辑与配置。

## 5. 用户与权限管理

### 5.1 用户管理

支持：

- 用户注册
- 用户登录
- 角色分配
- 创建用户
- 编辑用户
- 删除用户

用户角色至少包括：

- 标准用户
- 管理员
- 超级管理员

### 5.2 权限控制

采用 RBAC（基于角色的访问控制）。

权限至少包括：

- 知识问答权限
- 数据分析权限
- 报告生成权限

要求：

- 不同角色支持不同权限组合
- 管理功能仅管理员及超级管理员可访问
- 超级管理员拥有最高权限

### 5.3 数据统计与监控

管理后台首页展示系统运行状态及核心指标。

统计指标至少包括：

- 用户数
- 知识库数
- 文档数
- 切片数
- 报告模板数
- 报告生成记录数
- 知识问答次数

提供近 30 天趋势图：

- 知识问答趋势
- 文档上传趋势
- 报告生成趋势

支持快捷导航至各管理模块。

## 6. 非功能性需求

### 6.1 性能

- 文档解析与向量化处理支持后台异步执行。
- 知识问答支持流式输出。
- 报告生成支持分章节流式生成。

### 6.2 可用性

- 支持明暗主题切换。
- 支持主流现代浏览器。
- 提供明确的：

  - 加载状态
  - 错误提示
  - 空状态引导

### 6.3 安全性

- 所有 API 请求必须进行身份认证。
- 知识库及文档删除操作需二次确认。
- 系统设置等敏感功能仅授权管理员可访问。

### 6.4 可维护性

- 分段策略和检索策略支持动态调整。
- 调整后自动触发数据重处理。
- 大语言模型、嵌入模型、重排序模型支持运行时配置修改。
- 无需重启服务。
- 文档解析后端支持运行时切换。

### 6.5 扩展性

- 支持多知识库独立管理。
- 支持灵活组合检索。
- 报告生成框架支持扩展新的报告类型和模板。
